# Githab-finder

### Base for future React project with simple dependencies

### To run the project

- \$ git clone https://github.com/habin-isa/githab-finder.git
- \$ cd githab-finder
- Add 'config.json' file in /src with your personal access token (see src/example.config.json)
- \$ yarn
- \$ yarn start

### Examples contained

- Styled components
- Jest testing
- Proptypes
